Privacy Policy

Last updated: December 23, 2025

Manga Helper ("the Extension") is committed to protecting your privacy. This policy explains what data we collect and how we use it.

Data We Collect

IP-based Geolocation

The Extension sends a request to a third-party geolocation service to determine your country for displaying region-relevant advertisements. Your IP address is processed by this service but is not stored by the Extension. Only the resulting country code is used locally.

Local Cache

The Extension stores the following data locally on your device:

  • Manga metadata (titles, chapter lists) cached from supported platforms
  • Your reading progress on supported manga sites

User Profile (for Ads)

The Extension creates a local profile for ad targeting and recommendations:

  • Geolocation data: country, city, timezone, language preferences
  • Connection info: ISP name, VPN/proxy detection
  • Device info: platform, browser, screen resolution, hardware specs

This profile is stored locally and used to display relevant ads. It is never transmitted to external servers.

Usage Statistics (Local Only)

The Extension may collect the following statistics locally for recommendations and ad relevance:

  • Viewed manga titles and genres
  • Ad impressions and clicks

This data is stored locally on your device and is never transmitted to external servers.

Data We Do NOT Collect

  • Personal information (name, email, etc.)
  • Browsing history outside supported manga platforms
  • Authentication credentials

Third-Party Services

The Extension communicates with the following services:

  • Supported manga platforms (MangaLib, Senkuro, MangaBuff, ReadManga, Inkstory) — to fetch manga data and reading progress
  • IP geolocation service — to determine your country (IP is not stored by the Extension)
  • GitHub — to fetch extension configuration (no user data is sent)

Data Storage

All cached data is stored locally using the browser's storage API. You can clear this data at any time by:

  • Removing the Extension
  • Clearing browser extension data

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated revision date.

Contact

If you have questions about this Privacy Policy, please open an issue on our GitHub repository.
